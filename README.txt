To run this program you need to install Python3, OpenCV, NumPy and Matplotlib
You can run this program by typing this command in terminal : 

Python main.py {Path of the image to analyse}

If you don't put parameter to main.py it will try to analyse with the image called by default : 'Asian-hornet-WP.jpg'

The folder IA OpenCV refers to the abandon part of the project containing AI with Tensorflow.
